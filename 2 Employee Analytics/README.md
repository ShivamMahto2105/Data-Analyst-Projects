# 📊 Employee Analytics (SQL + Python)

## 📌 Project Overview
This project focuses on analyzing employee compensation and workforce data using **Python and SQL**. It simulates a real-world HR dataset and performs data analysis to extract insights such as salary distribution, department-wise trends, and attrition patterns.

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy, Matplotlib)
- SQL (Microsoft SQL Server)
- SQLAlchemy

---

## 📂 Dataset
The dataset consists of two relational tables:

- `employees.csv`
- `departments.csv`

📥 ![Dataset Link](https://github.com/ShivamMahto2105/Data-Analyst-Projects/blob/main/2%20Employee%20Analytics/dataset.py) : Generate the Dataset.
---

## 🗄️ Database Schema
- `employees` table  
- `departments` table  
- Relationship: `dept_id` (Foreign Key)

---

## 🔍 Exploratory Data Analysis (EDA)
Performed using Python:
- Data cleaning & validation  
- Salary distribution analysis  
- Gender pay gap analysis  
- Attrition trends  
- Department-wise insights

## 🧠 SQL Analysis
The project includes **15 SQL queries (Basic → Advanced)** covering:
- Filtering & aggregation  
- Joins

- Business insights

## 📈 Key Insights
- Salary varies significantly across departments  
- Higher experience leads to higher compensation  
- Attrition is higher in specific departments  
- Pay gap observed across gender groups
